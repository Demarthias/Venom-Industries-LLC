How It Works

One click.
Full picture.

SaaS Detective reads the signals every tool leaves behind — JavaScript globals, DOM structure, response headers, and script source paths — and surfaces them instantly in a clean popup.

Unlike BuiltWith, it doesn't send your browsing data to a server. Unlike Wappalyzer's free tier, it doesn't throttle results. It runs entirely in Chrome using only data from pages you've already loaded.

1

Visit any site

Navigate to any page — competitor, prospect, or client site.

2

Click the icon

SaaS Detective scans JS globals, DOM, headers, and script paths.

3

See the full stack

Every tool categorized, with detection reason shown for each finding.

// Detection logic sample — Stripe
// Stripe detection
{
  name: 'Stripe',
  category: 'Payments',
  signals: [
    {
      type: 'script-src',
      pattern: /js\.stripe\.com/,
      confidence: 1.0,
    },
    {
      type: 'window-global',
      key: 'Stripe',
      confidence: 0.95,
    }
  ]
}
