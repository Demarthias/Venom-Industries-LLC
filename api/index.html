Frequently Asked Questions

How does LagSlayer reduce ping? ‚ñº

LagSlayer applies modern TCP congestion control algorithms (BBR, CUBIC) and fair queueing (FQ_CODEL) to reduce buffer bloat and prioritize interactive traffic. This reduces ping variance and overall latency.

Is LagSlayer safe to use? ‚ñº

Yes. LagSlayer only modifies reversible kernel network parameters. It includes a one-command reset function that restores all settings to safe defaults. No drivers are installed and no permanent changes are made.

Which Linux distributions are supported? ‚ñº

LagSlayer supports Ubuntu (18.04+), Debian (10+), CentOS (7+), RHEL (7+), Arch Linux, and most derivatives. Kernel version 4.9+ required for full functionality.
